# Finance Dashboard Pro+

A modern financial analytics dashboard that enables users to upload, process, and analyze transaction data with real-time insights, predictive analytics, and anomaly detection.

## Overview

This project provides an end-to-end solution for personal finance tracking. Users can upload Excel/CSV files or store transactions in a database, and the system automatically cleans, analyzes, and visualizes the data.

## Key Features

* Data upload (Excel/CSV)
* Automatic data cleaning and validation
* Interactive financial dashboard with KPIs
* Expense breakdown and time-series analysis
* Filtering by date and transaction type
* Future spending prediction (machine learning)
* Anomaly detection for unusual transactions
* Downloadable processed data
* SQLite database integration

## Tech Stack

* Python
* Streamlit
* Pandas
* Scikit-learn
* SQLite
